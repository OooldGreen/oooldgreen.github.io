句型#

肯定句#

主宾谓： 私は　マイク・ミラーです。

否定句#

サントスさんは　学生 じゃありません。

疑问句#

  1. 助词か表示疑问，读升调

    ミラーさんは　会社員　ですが。

    • 肯定回答：はい、會社員です。
    • 否定回答：いいえ、学生じゃありません。

  2. 询问要做什么的疑问句

疑问词#

  • どんな　　什么样的

5W#

  • 何の　　有关什么的，表示内容
  • どうして　　为什么 　eg: どうして　日本語を　勉強しますか。
  • 誰（だれ）　 谁，哪位
  • どこ 哪里
  • いつ 什么时候

选择疑问句#

— それは　ボールです ですか、シャープペンシル　ですか。
— ボールペンです。

有动词的句式#

肯定形否定形
现在形〜ます〜ません
过去形〜ました〜ませんでした
  • 现在肯定：私は　毎朝　六時に　起きます。
  • 过去肯定：私は　昨日　勉強しました。

勉強します
勉強しました
勉強しません
勉強しませんでした

全否定句#

— 今朝　どこへ　行きましたか。
— どこ　「へ」も　行きませんでした。 哪里也没有去。

邀请#

  1. 否定型 + か、〜ませんか，邀请别人做某事，更为尊重

    — 一緒に　神戸へ　行きませんか。 要一起去神户吗？
    — 肯定回答：ええ、行きましょう。 好，去看吧。
    — 否定回答：すむません、ちょっと…

  2. 〜ましょう，积极的邀请，提议

    ちょっと　休みましょう。 休息一下吧
    終わりましょう 结束吧

助词#

も#

助词，也，表示同前

サントスさんも　会社員です。

の#

  1. 表示属性

    私は　IMCの社員です。

  2. 表示所产

    疑问使用「どこ/どちら」询问，后者更为郑重

    — これは　どこ車ですか。
    — これは　日本の車です。

  3. 表示内容

    — これは　何の本ですか。　这本书内容是什么？
    — カメラの本です。　　关于相机的书

に#

  1. 表示动作进行的时间点

    私は　六時に　起きます。

    私は　7月15日に　国へ　帰ります。

  2. 表示动作的到达点

    私は　学校に　行きます。

  3. 表示接触点

    友達に会います 和朋友碰面，见面

  4. 表示动作的对方

    私は　木村さんに　花を　あげます。　我送花给木村小姐。

    から（只有当前面是施（授）方的时候可以使用，前面的授方是组织的情况只用から）：
    私は　カリナさんから　チョコレートを　もらいました。
    私は　銀行から　お金を　借りました。

と#

  1. 表示并列关系

    休みは　土曜日　と　日曜日　です。

  2. 表示动作伙伴，“跟”

    私は　家族と　日本へ　きました。

から#

  1. 表示起点，和まで一起使用

    私は　九時から五時まで働きます。

    要用于继续性动作，不能用于瞬间性的动作
    例如：終わります，只能使用「に」

  2. 因为、所以，表示原因

    今日は　子供の　誕生日ですから、早く　帰ります。

  3. 表示动作的对方（见に）

    只有当前面是施（授）方的时候可以使用，前面的授方是组织的情况只用から

    私は　カリナさんから　チョコレートを　もらいました。
    私は　銀行から　お金を　借りました。

まで#

  1. 表示终点，和から一起使用

    昼休みは　十二時から一時までです。

へ#

做助词的时候读え（e），表示移动的方向

私は　は　京都へ　行きます。(へ　に 都是对的)

で#

  1. 表示交通工具

    私は　タクシーで　うちへ　帰ります。

    步行是动作，不算交通工具
    私は　歩いて　学校へ　行きます。

  2. 表示动作进行单位

    私は　一人で　学校へ　来ました。

  3. 表示动作进行地点

    私は　駅で　新聞を　買います。

  4. 表示方法、手段

    私は　バソコンで　映画を　見ます。　用电脑看
    これは　日本語で　何ですか。　这个用日语怎么说？

  5. 我有～，一般表示我有什么事情

    約束が　あります。 有约会

    东西一般用を
    約私は　車を　持っています。

を#

  1. 表示动作作用对象（把）

    私は　本を　読みます。
    水を　飲みます
    テレビを 　見ます
    たぼこを　吸います

  2. 我有～，一般表示我有什么东西

    私は　車を　持っています。

    事情一般用が　　 約束が　あります。

が#

  1. 表示逆接，但是，不过

    高い　ですが、いいです。　很贵但很好

  2. 表示焦点

私は　イタリア料理か　好きです。 我喜欢意大利料理。

私は　日本語を　勉強します。 动词，表示动作，用把
私は　日本語が　わかります。　动词，非动作，用焦点

指示词#

单独接名词
近自己これこの
近对方それその
远方あれあの
  • これは　辞書です。 　
  • はい、それは　辞書です。
指事物指地方指事物、地方、人（郑重/口语）这（那）样这（那）样的
近自己（こ）これこのこここちら/こっちこうこんな〜
近对方（そ）それそのそこそちら/そっちそうそんな〜
远方（あ）あれあのあそこあちら/あっちあああんな〜
疑问句（ど）どれどのどこどちら/どっちどうどんな〜

接续词#

  • それから　然后

    うちで　宿題を　します。それから　CDを　聞きます。

  • そして　　而且

    便利です。そして　綺麗です。 很方便，而且很干净

副词#

ときどき　有时候

もう　已经#

私は　もう　メールを　送りました。

ー　もう　昼ご飯を　食べました。
ー　はい、もう食べました。
ー　いいえ、まだです。 还没

形容词#

肯定形否定形
名词学生です学生じゃありません
な形容词綺麗です綺麗じゃありません
い形容词高いです高くないです

いいです 好 -> 否定形 -> よくないです　不好

な形容词后接名词用「な」、い形容词后直接接名词

な形容词#

桜は　きれいです。
桜は　綺麗な 花です。

好きです
嫌いです

い形容词#

い形容词都是「い」结尾

富士山は　高い　山です。
